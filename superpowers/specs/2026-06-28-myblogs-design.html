MyBlogs — 个人博客设计规范 ​

项目概述 ​

  • 项目名称: MyBlogs
  • 项目类型: 个人博客（技术 + 生活 + 兴趣综合型）
  • 面向用户: 博主本人（内容记录）+ 偶然路过的网友
  • 核心目标: 提供干净、快速的阅读体验，让博主专注于写作

标志性元素 ​

首页顶部打字机终端提示符，逐字打出 guest@myblogs:~$ cat README.md，然后显示个人介绍。

功能清单（MVP） ​

  1. 首页：打字机动画 + 文章列表
  2. 文章详情页：Markdown 渲染 + 代码高亮
  3. 关于页
  4. 暗色主题（默认，唯一主题）
  5. 响应式布局（手机可用）
  6. 文章按日期排序

约束条件 ​

  • 不做管理后台 — 文章直接写 .md 文件
  • 不做评论系统 — 保持纯静态
  • 不使用非必要 JS — 除了打字机动画和 VitePress 自带功能
  • 部署到 GitHub Pages，仓库 https://github.com/peterWjx/personal_blog

部署流程 ​

  1. 本地写 Markdown 文章 → 放入 docs/posts/
  2. git push origin main
  3. GitHub Actions 自动构建 → 部署到 GitHub Pages
