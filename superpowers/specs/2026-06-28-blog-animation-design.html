博客动画增强 — 设计规范 ​

项目概述 ​

  • 目标: 为 MyBlogs（Peter的博客）全站添加生动而贴合终端美学的动画效果
  • 技术选型: GSAP 3 + ScrollTrigger + TextPlugin
  • 设计原则: 动画服务于"暗色终端"主题身份，增强阅读体验而非干扰内容

技术架构 ​

依赖 ​

  • gsap ^3.12.0 — 动画引擎（含 ScrollTrigger、TextPlugin）

文件结构 ​

docs/.vitepress/theme/
├── index.ts                     # ✏️ 注册 GSAP 插件 + v-reveal 指令
├── style.css                    # ✏️ 删改旧动画，新增动画辅助样式
├── composables/
│   ├── useScrollReveal.ts       # ✨ ScrollTrigger 滚动渐现 composable
│   └── useTypewriter.ts         # ✨ GSAP 打字机 composable
├── directives/
│   └── vReveal.ts               # ✨ v-reveal 自定义指令
├── components/
│   ├── MatrixRain.vue           # ✨ Matrix 微光雨 Canvas 组件
│   ├── ScrollProgress.vue       # ✨ 阅读进度条
│   └── PageTransition.vue       # ✨ 页面切换过渡
└── animations/
    └── presets.ts               # ✨ 预设 GSAP 动画配置

docs/index.md                    # ✏️ 使用新 composable + MatrixRain

全局注册流程 ​

  1. index.ts 中导入 GSAP 插件，注册为 Vue 全局属性 $gsap
  2. 注册 v-reveal 自定义指令
  3. 可选在 config.ts 中无需变更——所有组件在 Markdown <script setup> 中按需引入

动画效果明细 ​

G1 · 页面切换过渡 ​

  • 实现: PageTransition.vue — VitePress 没有暴露 <router-view>，采用替代方案：监听路由变化，对 .VPDoc 容器做 GSAP fromTo
  • 效果: 内容区 opacity 0→1，y 15px→0，duration 0.25s，ease power2.out
  • 注意: 首次加载不触发过渡

G2 · 导航栏滚动阴影 ​

  • 实现: CSS 方案 — 用 scroll() 驱动的动画或监听 window.scrollY 给 .VPNavBar 动态添加 scrolled class
  • 效果: 滚动超过 20px 时，导航栏底部浮现 0 1px 8px rgba(0,0,0,0.4) 阴影
  • transition: 0.3s

G3 · 链接下划线滑动 ​

  • 实现: 纯 CSS — .vp-doc a 使用 ::after 伪元素 + transform: scaleX(0) → scaleX(1) on hover
  • 效果: 下划线从左侧滑入，颜色 --accent，替代现有 border-bottom: dashed

G4 · 元素滚动渐现 ​

  • 实现: v-reveal 指令 + useScrollReveal composable（底层用 ScrollTrigger）
  • 效果: 目标元素进入视口时，从 opacity: 0, y: 30px 渐现到原位
  • 配置: threshold: 0.15, duration: 0.5s, ease: power2.out
  • 用法: <h2 v-reveal>标题</h2> 或 <div v-reveal="{ y: 20 }">...</div>

G5 · 滚动进度条 ​

  • 实现: ScrollProgress.vue — position: fixed; top: 0; height: 2px; background: var(--accent)，ScrollTrigger 更新 scaleX
  • 效果: 极细琥珀色线横贯页面顶部，从左到右反映阅读进度
  • z-index: 高于导航栏

H1 · GSAP 重写打字机 ​

  • 实现: useTypewriter composable — 接收目标字符串，返回 displayText（ref）
  • 效果: 逐字打印，每个字符间隔 50-90ms 随机微延迟，光标平滑闪烁
  • 替代: 现有 setInterval 实现

H2 · 个人介绍交错浮现 ​

  • 实现: <div v-reveal="{ stagger: 0.1 }"> 包裹介绍区域各子元素
  • 效果: 打字完成后内容显示，名字→描述→列表→footer 依次从下方滑入，每个延迟 0.1s

H3 · 文章列表项 hover 抬起 ​

  • 实现: 纯 CSS — .post-list li:hover 添加 transform: translateY(-2px) + 左侧 ::before 伪元素琥珀色竖线
  • 效果: 悬浮行轻微上浮，左侧出现 2px solid var(--accent) 指示器

P1 · 标题入场 ​

  • 实现: v-reveal 指令作用于 .vp-doc h1, .vp-doc h2, .vp-doc h3
  • 效果: 标题从下方 20px 滑入，h1/h2/h3 分别延迟 0/0.05/0.1s（通过 CSS 或指令参数）

P2 · 代码块终端窗口 ​

  • 实现: 纯 CSS — div[class*='language-'] 添加 ::before 三色圆点（红 #FF5F57 / 黄 #FEBC2E / 绿 #28C840）
  • 效果: hover 时代码块 transform: scale(1.01)，transition: 0.2s

P3 · Blockquote 滑入 ​

  • 实现: v-reveal 指令作用于 blockquote，from x: -20px
  • 效果: 引用块从左侧滑入，左边框先出现

X1 · Matrix 微光雨 ​

  • 实现: <MatrixRain> Vue 组件，Canvas 2D 绘制
  • 字符集: 键盘符号 {}[]()/<>$*#@. + 少量十六进制数字
  • 颜色: #7EB87E（终端绿），主体透明度 0.06-0.10，头部字符 0.15
  • 速度: 0.3-0.8 px/frame 随机，偶尔停顿
  • 密度: 10-12 列（移动端减半）
  • 性能: IntersectionObserver — 页面不可见时暂停 requestAnimationFrame
  • 位置: 首页 position: fixed; z-index: -1，不影响交互

性能约束 ​

  1. 所有 ScrollTrigger 实例在组件 onUnmounted 时 kill()
  2. Matrix 雨在视口不可见时暂停（IntersectionObserver）
  3. 移动端 Matrix 密度减半至 5-6 列
  4. GSAP 按需引入：gsap + ScrollTrigger + TextPlugin
  5. prefers-reduced-motion 时降级所有动画为静态

回滚策略 ​

  • GSAP 和 MatrixRain 作为可选增强，出错时不影响页面正常渲染
  • v-reveal 指令在 JS 失败时元素保持可见（opacity: 1 作为 fallback）
  • 可通过注释 index.ts 中的插件注册行快速禁用全部动画
